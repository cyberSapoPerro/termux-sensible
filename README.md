# termux_sensible

This script automate Termux setup.

## Features

- **Package installation**: Installs essential tools (neovim, zsh, starship, fzf, tmux, and more)
- **Nerd Fonts**: Downloads and installs Hack Nerd Font Mono for icon support
- **Terminal colors**: Tokyo Night-inspired dark color scheme
- **Zsh configuration**: Pre-configured with antigen plugin manager and popular plugins
- **Shell integrations**: Starship prompt, zoxide, and atuin history sync

## Quick Start

Clone and run:

```bash
git clone https://github.com/cyberSapoPerro/termux-sensible.git
cd termux_sensible
bash run.sh
```
